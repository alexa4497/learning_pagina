Variables

Las varibales en la programaci贸n son utilizadas para almacenar datos. su estructura es el nombre de la variable junto con el valor que le asignamos. Las variables pueden almacenar disintos tipos de datos como numericos, numeros decimales, palabras o booleanos. Utilizamos el signo igual "=" para indicar el valor de la variable

Para asignarle un nombre a una variable podemos usar letras, numeros y el caracter de guion bajo o barra baja.

ejemplo:

edad = 17
aprendiz = true
nivel = 2.20

como podemos observar creamos cuatro variables, a la variable nombre le asignamos una palabra, a la variable edad le asignamos un numero entero, a la variable aprendiz le damos un valor verdadero y por ultimo a la variable nivel le asignamos un valor decimal.

Un dato curioso es que las variables son mutables es decir que se puede cambiar su valor y podemos unsar la funci贸n print para imprimirlas en consola.

ejemplo:

5

Como pudiste observar creamos una variable llamada numero, le asignamos el valor numerico de 1 y posteriormente cambiamos su valor a 5, python imprime el valor mas reciente aignado a la variable.
