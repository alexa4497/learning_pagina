Introduccion a python

Python es un lenguaje de progrmación de alto nivel con usos a nivel general , python es un lenguaje bastante popular gracias a su simplicidad y amplios campos de uso.


Python fue creado por Guido Van Rossum un programador nativo los Países Bajos.Nació el 31 de enero de 1956

📀 Características

  • Es un lenguaje de alto nivel, las lineas de codigo de python son similares a las del lenguaje de python.
  • Es un lenguaje interpretado es decir que las lineas de codigo se ejecutan linea por linea
  • Es un lenguaje tipeado dinámicamente, gracias a esto la escritura de codigo es mas rapida
  • Es un lenguaje facil de usar, su programacion es sencilla

📀 Ventajas

  1. Es un lenguaje de fácil comprensión con una sintaxis de sencilla lectura
  2. Es un lenguaje ligero ya que se puede escribir codigo en menos lineas que otros programas
  3. Python cuanta con infinidad de librerias las cuales facilitan la escritura de codigo para los programadores
  4. Al ser tan popular exiten muchos recursos de apoyo del programa como videos, tutoriales, foros, blogs, etc.
  5. Es un lenguaje gratis y de codigo abierto
  6. Tiene una gran comunidad al rededor del mundo y se puede contar con el apoyo de infinidad de programadores.
  7. Tiene un gran campo de aplicacion como lo es el desarrollo web,desarrollo de videojuegos, Análisis de Big Data entre otros

📀 Desventajas

  1. debido a la gran versatilidad de python llega a tener un procesamiento lento ya que no tiene un uso definido
  2. A pesar de ser un lenguaje de uso general en el area del desarrollo de aplicaciones moviles no es la mejor opcion , existen lengaujes mas especializados para esta tarea.

📀 Instalación de python

  1. Dirijase a el siguiente enlace que lo llevara a la pagina oficial de python e instale python
    2. https://www.python.org/downloads/
  2. Para verificar que la instalación fue correcta dirijase a su terminal o simbolo de sistema y escriba la siguiente linea de codigo
    3. >>> python -V
  3. Si la instalación fue correcta se debera mostrar la versión instalada de python
    4. >>> python 3.11.1
