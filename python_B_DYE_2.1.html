Comentarios

los comentarios en los lenguajes de programación son las fracciones de texto de nuestro codigo las cuales son ignorados por el lenguaje de programación, con los comentarios podemos organizar nuestro codigo por secciones o podemos separar ciertas lineas de codigo las cuales no queremos que se ejecuten junto con el programa, para identificar un comentario en python utilizamos el simbolo "#" el cual indica el inicio de nuestro comentario.

ejemplo:
# este es un comentario de python
print("saludos✌")
resultado:
saludos✌

Como puedes observar en la consola solo se imprime la linea de codigo y python pasa por alto el comentario.

Comentarios multiples

Es importante saber que seolo se identifica como comentario a el contenido posterior al simbolo "#" es decir que podemos convertir una linea de codigo normal a comentario y viceversa segun nuestra conveniencia.

ejemplo:
#print("esta es una linea de codigo convertida en comentario")
#comentario nuemro dos
#comentario nuemro tres
resultado:

la consola no impirme nada ya que nuestro codigo son comentarios.

recordatorios para terminar estas ultimas dos lecciones
  • print es una función usada para imprimir mensajes en la terminal
  • python ejecuta una linea de codigo al tiempo
  • para identificar un comentario utilizamos el simbolo #
