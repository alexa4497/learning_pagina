Tipos de datos

📀 Int

el tipo de dato int son los numeros enteros, los datos de tipo int no contien decimales, solo numeros enteros, incluyen el numero 0 y tambien todos los numeros positivos y negativos enteros.Como por el ejemplo el numero de personas en un lugar.

edad = 17
numero_de_hermanos = 3

📀 Float

Los datos de tipo float son numeros decimales, los podemos utilizar para calcular mediciones no enteras o fracciones impropias como por ejemplo la altura de una persona.

pi = 3.1415
altura = 1.72

📀 string

Los datos de tipo string son un tipo de dato para almacenar cadenas de texto, para señalar un string debemos colocar el texto entre dos comillas dobles " " o dentro de dos comillas simples ' '.

Hay que tener en cuenta de que un numero ya sea decimal o entero puede ser identificado como string si lo encerramos entre dos comillas.

usuario = "alex_45"
fruta = "limon 🍋"

📀 Booleano

Los datos de tipo booleano son valores que solo se pueden ser falsos o verdaderos True o False.

estamos_en_año_1950 = False
