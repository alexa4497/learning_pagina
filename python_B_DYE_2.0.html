Tipos de datos y estructuras de python

En python existen diferentes tipos de datos los cuales almacenan cierto tipo de información o tienen diferentes funciones.Estos tipos de datos pueden ser enteros, numeros reales, booleanos entre otros.Continua aprendiendo y paso a paso te enseñare cada uno de ellos y sus funciones.

Print()

Como lo vimos en la lección pasada la función print imprime los valores que indiquemos, su sintaxis se basa en colocar la palabra print y entre dos comillas dobles o comillas simples indicar que queremos que se imprima o muestre en consola.

ejemplo:
print ("hola💖")
resultado:
hola💖
segundo ejemplo:
print ('comillas simples')
resultado:
comillas simples

📀 Patrones print()

la funcion print tambien puede ser usada multiples veces, es decir una seguida de la otra como lo veras a continuación

ejemplo:
print("me llamo Alex♊")
print("tengo 17 años")
print("mi fruta favorita es la manzana🍎")
resultado:
me llamo Alex♊
tengo 17 años
mi fruta favorita es la manzana🍎
